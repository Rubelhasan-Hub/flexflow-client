# FLEXFLOW - Fitness & Gym Management Platform

Welcome to Flexflow , a comprehensive platform designed for fitness enthusiasts, trainers, and administrators to streamline gym management and community engagement.

## 🚀 Live Demo
https://flexflow-gym.vercel.app


## 📋 Purpose
FlexFlow is a modern, full-stack web application designed to bridge the gap between fitness trainers and health enthusiasts. It provides a seamless platform for class scheduling, professional booking management, and interactive community engagement.

## ✨ Key Features
* **Dynamic Booking System: Enables users to browse and book professional training sessions with real-time availability.
* **Role-Based Access Control: Secure authentication paths for Admins, Trainers, and Users.
* **Interactive Community Forum: A dedicated space for members to share knowledge, post updates, and engage through like/dislike interactions.
* **Responsive Dashboard: A clean, user-friendly interface optimized for all devices to ensure a smooth user experience.
* **Secure Authentication:** Powered by Better Auth with Credential and Google Login.

## 🛠 Tech Stack
* **Frontend:** Next.js, Tailwind CSS, Framer Motion, HeroUI
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** Better Auth (JWT-based)
* **Payments:** Stripe API
* **Other:** ImgBB (Image Hosting)

## 💡 How to Run Locally
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up your environment variables.
4. Run the development server: `npm run dev`
